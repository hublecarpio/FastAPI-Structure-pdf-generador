Puedes usar sintaxis Jinja2 en tus plantillas HTML:

{{ "{{ variable }}" }} - Insertar valor de variable
{{ "{% for item in items %}" }} - Bucle
{{ "{% if condition %}" }} - Condicional
{{ "{{ item.property }}" }} - Acceder a propiedades

Ejemplo:

<h1>Factura #{{ "{{ invoice_number }}" }}</h1>
