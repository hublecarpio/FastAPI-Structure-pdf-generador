Como usar las API Keys

Usa tu API Key en el header de autorizacion para acceder a la API programaticamente:

Authorization: Bearer TU_API_KEY

Ejemplo con curl:

curl -X POST "/render/TEMPLATE_ID" \
     -H "Authorization: Bearer TU_API_KEY" \
     -H "Content-Type: application/json" \
     -d '{"nombre": "Juan"}' \
     --output documento.pdf
